"# FoodieHub - Веб-сообщество для гастрономов" 
"## Описание проекта" 
"Платформа для обмена рецептами, поиска ресторанов и организации кулинарных мероприятий" 
"## Установка" 
"1. Клонировать репозиторий: \`git clone https://github.com/YuriDokill/FoodieHub.git\`" 
"2. Установить зависимости: \`composer install\`" 
"3. Настроить БД в \`config/db.php\`" 
"4. Применить миграции: \`yii migrate\`"
